## Education Course Seller Website

This repository contains the source code for an educational course seller website. The website is built using HTML, CSS, Bootstrap, and JavaScript. It is completely responsive.

### Features:

- **Responsive Design:** The website includes 6-7 responsive pages optimized for various devices, including mobile phones, tablets, and desktops.
  
- **Embedded YouTube Videos:** The course details page includes embedded YouTube videos that are responsive and adjust to different screen sizes for optimal viewing experience.
  
- **Language Change:** The course details page features a language change functionality allowing users to toggle between different languages for course descriptions or other relevant content.

### Pages:

1. **Home Page:** Landing page introducing the website and its services.
2. **About Page:** Information about the company or platform offering the courses.
3. **Courses Page:** Listings of available courses for sale.
4. **Course Details Page:** Detailed information about a specific course, including embedded YouTube videos and language change functionality.
5. **Trainers Page:** Information about the trainers or instructors offering the courses.
6. **Contact Page:** Contact form or information for users to get in touch with the website administrators.

### Usage:

1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Navigate to the respective HTML files to view and modify the code.
4. Customize the content, styles, and functionality as needed.
5. Test the website on different devices and screen sizes to ensure responsiveness.
6. Deploy the website to a web server or hosting platform for public access.
